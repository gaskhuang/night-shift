# 🔄 每日迭代與行動推薦系統

> 「不是每天做更多事，而是每天做更正確的事」

## 系統概述

這個系統與**夜班系統**搭配運作：
- 🌙 **夜班 (23:00-06:00)**：AI 自動執行背景任務
- ☀️ **日班 (08:00-22:00)**：基於夜班成果 + 動態優先級，推薦你下一步行動

---

## 📊 核心機制

### 1. 每日循環 (Daily Loop)

```
06:00 夜班結束 → 生成晨報
08:00 G大起床  → 閱讀晨報
      ↓
22:00 覆盤時間  → 記錄今日成果 + 明日意圖
      ↓
23:00 夜班開始  → 基於覆盤調整任務優先級
```

### 2. 三層決策架構

| 層級 | 頻率 | 內容 | 文件位置 |
|------|------|------|----------|
| **戰術層** | 每日 | 今天做什麼、優先級排序 | `memory/YYYY-MM-DD.md` |
| **策略層** | 每週 | 本週目標、資源分配 | `memory/WEEK-XX.md` |
| **願景層** | 每月 | 月度主題、重大決策 | `memory/MONTH-YYYY-MM.md` |

---

## 🎯 下一步行動推薦引擎

### 推薦邏輯 (Scoring Algorithm)

每個潛在行動會被打分，最高分者成為「推薦下一步」：

```
總分 = (緊急度 × 0.3) + (影響力 × 0.4) + (可行性 × 0.2) + (能量匹配 × 0.1)
```

| 維度 | 說明 | 評分標準 |
|------|------|----------|
| **緊急度** | 時間壓力 | 1-5 (5=今天不做會出事) |
| **影響力** | 變現或戰略價值 | 1-5 (5=直接影響收入) |
| **可行性** | 今天能完成多少 | 1-5 (5=2小時內可完成) |
| **能量匹配** | 你現在的狀態 | 1-5 (5=完全匹配當下能量) |

### 行動類型分類

| 類型 | 適合時機 | 例子 |
|------|----------|------|
| 🚀 **火箭型** | 能量高、需要深度工作 | 寫核心程式、戰略規劃 |
| 🔄 **維護型** | 能量中、日常運作 | 回覆訊息、整理檔案 |
| ⚡ **快速型** | 能量低、碎片時間 | 審批文件、小決策 |
| 🌱 **成長型** | 任何時候、長期投資 | 學習新技術、建立關係 |

## 📝 每日覆盤格式 (Evening Review)

每天 22:00 記錄於 `memory/YYYY-MM-DD_Evening.md`：

```markdown
# 🌆 每日覆盤 - 2026-03-07

## ✅ 今日完成 (Done)
- [x] 建立夜班系統
- [x] 完成 X 篇內容
- [x] ...

## 📊 今日數據
- 完成任務數: X
- 專注時長: X 小時
- 能量水平 (1-10): X
- 滿意度 (1-10): X

## 💡 今日洞察
- 發現什麼新機會？
- 學到什麼教訓？
- 有什麼意外收穫？

## 🎯 明日意圖 (Tomorrow Intent)
如果我明天只能做一件事，那會是：_________

期望能量狀態：高/中/低
期望專注時段：上午/下午/晚上

## 🔥 待孵化想法
- 臨時想到但沒時間做的點子
- 需要進一步思考的方向
```

## 🤖 行動推薦流程

### 每天早上 08:30 自動生成：

```
# 🎯 今日行動推薦 - 2026-03-08

## 📰 夜班成果摘要
(從晨報提取)

## 🏆 TOP 3 推薦行動

### #1 [最高分行動]
**行動**: 具體做什麼
**原因**: 為什麼現在推薦這個
**預估時間**: X 分鐘/小時
**預期成果**: 完成後會得到什麼
**風險**: 可能遇到的阻礙

### #2 [次選]
...

### #3 [備選]
...

## 🔄 如果能量不足...
- 低能量選項 A
- 低能量選項 B

## 📅 今日時間塊建議
- 09:00-11:00: 深度工作 (火箭型任務)
- 14:00-16:00: 協作/會議
- 20:00-21:00: 覆盤與規劃
```

## 🚀 啟動方式

### 自動化 (推薦)
```bash
# 每天早上 08:30 生成行動推薦
30 8 * * * /Users/user/night-shift/daily-iteration/next_action_engine.py

# 每天晚上 22:00 提醒覆盤
0 22 * * * /Users/user/night-shift/daily-iteration/daily_review.sh
```

### 手動觸發
```bash
# 獲取今日行動推薦
/Users/user/night-shift/daily-iteration/next_action_engine.py --today

# 開始每日覆盤
/Users/user/night-shift/daily-iteration/daily_review.sh

# 根據當前能量獲取推薦
/Users/user/night-shift/daily-iteration/next_action_engine.py --energy high
```

## 🎮 與現有系統整合

### 輸入來源
1. **夜班晨報** → 已完成工作的基礎
2. **TASK_LIST.md** → 待辦任務池
3. **每日覆盤** → 能量狀態與明日意圖
4. **長期目標** (MONTH/WEEK) → 策略方向

### 輸出去向
1. **console/notification** → 即時推薦
2. **recommendation.md** → 書面記錄
3. **memory/YYYY-MM-DD.md** → 整合到日記
